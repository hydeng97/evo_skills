# articles

This directory stores normalized source-material packages.

Each package should use the shape:

```text
articles/<source_id>/
  source_meta.json
  distilled.md
  examples.md
  coach_notes.md
```

- `source_meta.json`: source title, author, type, date, topic tags, and rights notes
- `distilled.md`: core thesis, concepts, mechanisms, boundaries, and common misuse
- `examples.md`: worked examples, analogies, and application scenarios
- `coach_notes.md`: scenario prompts, intervention ideas, and coaching angles

This directory is for processed content, not raw private memory.
